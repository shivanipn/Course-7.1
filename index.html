Arithmetic Calculator

JavaScript makes websites interactive by responding to user actions like clicks or mouse movements. It runs in web browsers like Chrome or Safari and may need to be enabled. For example, it can validate form inputs or handle events like button clicks and link navigation.
To create an Arithmetic Calculator in JavaScript, you will need to:
1. Create JavaScript function for each arithmetic operator (+, -,*,/) such as

üî¢ Decimal to Binary Converter

This tool converts decimal numbers (base 10) into their binary equivalents (base 2) using JavaScript.

üß† Concept
  • Decimal system: Uses digits 0‚Äì9.
  • Binary system: Uses only 0 and 1.
  • Conversion logic: Repeatedly divide the decimal number by 2 and store remainders. Read remainders in reverse to get binary.
‚öôÔ∏è How It Works
  • dec_to_bi() ‚Äì Convert Decimal to Binary
  • Gets input from a textbox (txt1).
  • Repeats division by 2, stores remainders in an array.
  • Reverses array to form binary result.
  • Displays output in the textbox with ID output.
OutPut: 
             function dec_to_bi() {
              let input_val = parseInt(document.getElementById("txt1").value);
              let arr = [];
              while (input_val !== 0) {
                arr.push(input_val % 2);
                input_val = parseInt(input_val / 2);
              }
              let res = arr.reverse().join('');
              document.getElementById("output").value = res;
            }

üïí Time Converter

The Time Converter is a simple JavaScript tool that allows users to convert between hours, minutes, and seconds. It automatically calculates equivalent values based on user input in one unit field and displays the converted results in the other two fields.

üöÄ How It Works

üõ†Ô∏è Conversion Function: calculate_time()

  • Gets values from the input fields: hour, minute, second
  • Checks that only one field is filled at a time.
  • Performs conversion based on the input using formulas.
üî¢ Input üîÑ Converted To
Hours Minutes = hours √ó 60
Seconds = hours √ó 3600
Minutes Hours = minutes √∑ 60
Seconds = minutes √ó 60
Seconds Hours = seconds √∑ 3600
Minutes = seconds √∑ 60

üì§ The results are automatically displayed in the respective output fields after calculation.

Reset Function: reset_value()
  • Clears all input fields to reset the form.
  • Called when the user clicks the "Reset" button.
üß™ Key JavaScript Code 
  
  function calculate_time() {
    let hour = document.getElementById("hour").value;
    let minute = document.getElementById("minute").value;
    let second = document.getElementById("second").value;
  
    if ((hour !== "" && minute !== "") || 
        (minute !== "" && second !== "") || 
        (hour !== "" && second !== "")) {
      alert("You can only enter a value in one textbox at a time.");
    } else if (hour !== "") {
      document.getElementById("minute").value = hour * 60;
      document.getElementById("second").value = hour * 3600;
    } else if (minute !== "") {
      document.getElementById("hour").value = (minute / 60).toFixed(2);
      document.getElementById("second").value = minute * 60;
    } else if (second !== "") {
      document.getElementById("hour").value = (second / 3600).toFixed(2);
      document.getElementById("minute").value = (second / 60).toFixed(2);
    }
  }
  
  function reset_value() {
    document.getElementById("time_convert").reset();
  }

  • A "Reset" button to clear all values
