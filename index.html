üí° Intellectual Property Rights (IPR)

Intellectual Property (IP) refers to creations of the mind like inventions, designs, literary works, and symbols. IP Rights (IPRs) protect these creations, allowing innovators to gain recognition or financial benefits. Key IPR types include:

  • Trademark: A unique symbol, word, or design that identifies products/services. Valid for 10 years.
  • Patent: Grants rights to inventors to exclude others from using their invention for up to 20 years.
  • Copyright: Protects creative works like books, music, software, and art, allowing exclusive rights to reproduce or display them.

Software Piracy

Software piracy refers to the illegal copying, sharing, or installation of software without proper licensing. When you purchase software, you're buying a license to use it, not the software itself.
Types of software piracy include:

  • Copying
  • Downloading
  • Uploading
  • Sharing or selling unauthorized copies

Why it's unethical: It harms developers who invest time and resources in creating software, leading to increased prices for legal users.
Tip: Always read the software‚Äôs "read me" file to understand licensing terms.

Plagiarism

Plagiarism is using someone else's words, ideas, or work without giving credit. It includes copying from books, websites, or digital media and presenting it as your own.
Acts considered plagiarism:

  • Copying others‚Äô work without acknowledgment
  • Failing to use quotation marks
  • Citing sources incorrectly
  • Paraphrasing too closely
  • Only citing part of a source

To avoid plagiarism: Use your own words, quote properly, and give full credit to original authors.

üî• Firewall

Definition

A firewall is a network security system that monitors and controls incoming and outgoing network traffic based on a set of predefined security rules. It acts as a protective barrier between a trusted internal network and an untrusted external network (like the Internet).

üõ°Ô∏è Purpose of a Firewall
  • Blocks unauthorized access
  • Filters traffic based on rules
  • Protects internal networks from external threats

üß∞ Types of Firewalls

1. Hardware Firewall
  • A physical device usually integrated with routers.
  • Suitable for large networks and offers network-wide protection.
  • Filters traffic before it reaches individual devices.
2. Software Firewall
  • Installed on individual computers like any other software.
  • Often pre-installed with operating systems (e.g., Windows Defender Firewall).
  • Monitors and controls applications trying to access the internet.
  • Allows users to customize settings based on their security needs.

Intrusion Detection System (IDS)

An Intrusion Detection System (IDS) is a security tool designed to detect suspicious activities early to prevent intrusions. It monitors incoming and outgoing network traffic for patterns indicating potential attacks, alerting administrators to possible threats or policy violations. IDS works by analyzing system vulnerabilities, checking file integrity, and identifying known attack patterns. It can also monitor the internet for new threats. Examples of open-source IDS tools include ACARM-ng, AIDE, and Bro NIDS.

Data Encryption / Cryptography

Data encryption is a security technique used to protect sensitive information by converting it into an unreadable format called ciphertext. This ensures that only authorized individuals, who have the correct decryption key, can access the original data. Encryption is essential for safeguarding information such as passwords, credit card numbers, and personal details, especially during online transactions.
When you perform activities like online banking or shopping, your information is encrypted before being sent over the internet. At the receiving end, it is decrypted using the appropriate key to reveal the original content. Without the key, no one can interpret the encrypted data, making it highly secure.

There are two main types of encryption:
  1. Symmetric Encryption ‚Äì The same key is used for both encryption and decryption. It is fast and efficient but requires both parties to share the key securely.
  2. Asymmetric Encryption (Public Key Encryption) ‚Äì Uses a pair of keys: a public key to encrypt and a private key to decrypt. It is more secure for communication over open networks.

Cryptography

Cryptography is the broader science and art of encoding information to protect its confidentiality and authenticity. It involves converting plaintext into ciphertext so that only the intended recipient can understand the message.

The key goals of cryptography are:
  • Confidentiality ‚Äì Ensuring only the intended person can read the message.
  • Integrity ‚Äì Ensuring the message has not been altered.
  • Authentication ‚Äì Verifying the identity of the sender.
  • Authentication ‚Äì Ensuring the sender cannot deny having sent the message.

Backup

Backup is the process of copying important data to a secondary storage location to protect it from loss due to accidental deletion, system crashes, theft, or cyber-attacks. Backups help recover data if the original files are damaged or lost. Common backup storage options include external hard drives, USB drives, optical disks, and cloud storage services like Google Drive, Dropbox, and OneDrive.
