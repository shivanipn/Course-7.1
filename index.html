“Introduction to JavaScript”

A Brief History of JavaScript

JavaScript is a language that adds interactivity to websites. Created in the 1990s, it’s now built into most devices. Web developers use HTML for content, CSS for design, and JavaScript for behavior.

About JavaScript

JavaScript makes websites interactive by responding to user actions like clicks or mouse movements. It runs in web browsers like Chrome or Safari and may need to be enabled. For example, it can validate form inputs or handle events like button clicks and link navigation.

Uses of JavaScript

JavaScript is used in web pages for several purposes. Some of these are:

  • Changing the appearance of HTML elements dynamically.
  • Responding to user interactions such as mouse clicks.
  • Creating animations.
  • Validating user inputs.
Case Sensitivity in JavaScript

JavaScript is case-sensitive, so keywords, variables, and functions must use consistent capitalization. For example, "while" must be written exactly as "while," not "While" or "WHILE."

Structue and Syntax

JavaScript can be implemented using JavaScript statements that are placed within the <script>... </script> HTML tags in a web page.
JavaScript code is placed inside <script> tags, usually within the <head> section of a web page. These tags tell the browser to treat the enclosed text as a script.
<script>
  JavaScript code
</script>

Your First JavaScript Code

Let us take a sample example to print out "Hello World". We call a function document.write() which writes a string into our HTML document. A JavaScript string stores a series of characters like "umbrellatree Books".

Whitspace and Line Breaks

JavaScript ignores spaces, tabs and newlines that appear in JavaScript programs. You can use spaces, tabs and newlines freely in your program and you are free to format and indent your programs in a neat and consistent way that makes the code easy to read and understand.

Semicolons are Optional

In JavaScript, simple statements usually end with a semicolon (`;`), like in C, C++, and Java. However, semicolons can be omitted if each statement is on a separate line.

Comment In JavaScript

JavaScript supports both single-line and multi-line comments.

  • Any text started with a 7/' is treated as a single-line comment and is ignored by JavaScript.
  • Any text between the characters /* and */ is treated as a multi-line comment.
The following example shows how to use comments in JavaScript. <script type="text/javascript"> // This is a single-line comment. /* This is a multiline comment in JavaScript It is very similar to comments in C Programming */ </script>
JavaScript External in File

If you are using identical JavaScript code on multiple pages of a site, the script tag provides a mechanism to allow you to store JavaScript in an external file and then include it into your HTML files. Here is an example to show how you can include an external JavaScript file "index.js" in your HTML code using script tag and its src attribute. 

<!DOCTYPE html>
<html>
<head>
  <!-- Linking external JavaScript file -->
  <script type="text/javascript" src="index.js"></script>
</head>
<body>

  <h1>Hello, world!</h1>

</body>
</html>

Defining and Using Variables

A variable is like a named container used to store data that a program can use and modify later.

  • It holds a value (like a number, string, or object).
  • Instead of using the value directly, you use the variable name to refer to it.
  • This makes your code more readable and flexible.

JavaScript Identifires

All JavaScript variables must be identified with unique names. These unique names are called identifiers.
The general rules for constructing names for variables (unique identifiers) are:

  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter, $ or _.
  • Names are case-sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names
Note: JavaScript identifiers or variable names are case-sensitive.
One Statement Many Variable

You can declare multiple variables in a single line by:

  • Starting with var, let, or const
  • Separating each variable with a comma

var person = "John Doe", carName = "Volvo", price = 200;
This declares three variables in one line, each with its own value.
A variable can be declared without assigning a value. In that case, it automatically gets the value undefined.
var carName; console.log(carName);
This is useful when the value will be calculated or provided later, such as user input.

JavaScript Variable Scope

JavaScript variables have 3 types of scope:

  • Block scope
  • Function scope
  • Global scope
Block Scope

ES6 introduced two important new JavaScript keywords: let and const. These two keywords provide Block Scope in JavaScript. Variables declared inside a { } block cannot be accessed from outside the block:

{ let x = 2; } // x can NOT be used here
{ var x = 2; } // x CAN be used here
Local Scope

Variables declared within a JavaScript function, are LOCAL to the function:
// code here can NOT use carName function myFunction() { let carName = "Volvo";
// code here CAN use carName } // code here can NOT use carName

Function Scope

JavaScript has function scope: Each function creates a new scope. Variables defined inside a function are not accessible (visible) from outside the function. Variables declared with var, let and const are quite similar when declared inside a function.

JavaScript Data Types

In JavaScript, the data type of a value defines the kind of data it holds. Variables can store different types of data like numbers, strings, and Booleans. In this lesson, we’ll focus on these three basic types:

  • Number: Represents numeric values (e.g., 10, 3.14, -5).
  • String: Represents text, written in quotes (e.g., "Hello", 'World').
  • Boolean: Represents logical values: true or false.
NUMBERS
  • JavaScript does not distinguish between integers and floating-point (decimal) numbers.
  • All numbers are stored as floating-point values, even if they look like whole numbers.
  • This is because JavaScript is untyped, meaning it handles the number type automatically at runtime.
Integer Values

let x = 0;
let x = 3;
let x = 10000000;

Floating-Point Values
  • Numbers that contain a decimal point.
  • Represented as: integral part + decimal point + fractional part.
  • Examples:

    • let x = 3.14;
    let x = 2345.789;

STRINGS
  • In JavaScript, strings represent alphanumeric data (text).
  • A string is one or more characters enclosed in single ('') or double quotes ("").
  • If a string starts with a single quote, it must end with a single quote. The same applies to double quotes.
Examples:

let s1 = 'This is a simple string'; // Single quotes
let s2 = "This is also a simple string"; // Double quotes

  • Numbers in quotes (like "123") are also considered strings, not numbers:
  • Examples:

    • let s3 = "123"; // This is a string, not a number

BOOLEAN

BOOLEAN: TRUE, FALSE It's the namesake of the mathematician George Bool. Booleans only have two values: true and false. As you'll come to know, these are important values when it comes to adding logic to our programs.

JavaScript Operators

Operators are symbols used to perform actions on data, like numbers, strings, or Boolean values. They help us solve problems and build logic in our code.
Types of JavaScript Operators:

  • Arithmetic Operators – Perform math operations (+, -, *, /)
  • Comparison Operators – Compare values (>, <, ==, ===)
  • Logical Operators – Combine true/false values (&&, ||, !)
  • Assignment Operators – Assign values (=, +=, -=)
  • Conditional (Ternary) Operator – Makes quick decisions (condition ? true : false)
  • Bitwise Operators – Work at the binary level (&, |, ^, ~)
Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations. Here are the common ones: These operators are commonly used in calculations, loops, and many other areas in JavaScript programming.

Comparison Operators

Comparison operators are used to compare two values. They return a Boolean value: true or false.

Logical Operators

Logical operators are used to combine multiple conditions or Boolean values. They return true or false based on the logic applied.

Conditional Operators

The conditional operator, also known as the ternary operator, allows you to make quick decisions based on a condition. It's a shorthand for an if...else statement.

Bitwise Operators

Bitwise operators work at the binary level, manipulating the bits (0s and 1s) of numbers. These are mostly used in low-level programming, optimization, and graphics.

Assignment Operators

Assignment operators assign values to variables. The basic one is =, but shorthand versions like +=, -=, and *= combine assignment with math operations for cleaner code.

JavaScript Array

An array in JavaScript is a special variable that can hold multiple values in a single variable. Think of it like a list of items.
Syntax: let fruits = ["Apple", "Banana", "Mango"];

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).
A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().
Function names can contain letters, digits, underscores, and dollar signs (same rules as variables).
// Function to compute the product of p1 and p2 function myFunction(p1, p2) { return p1 * p2; }

Dialog Boxes

Dialogue boxes are a kind of popup notification, this kind of informative functionality is used to show success, failure, or any particular/important notification to the user.
JavaScript uses 3 kinds of dialog boxes:

  • Alert
  • Prompt
  • Confirm

Alert Box

An alert box is often used if you want to make sure information comes through to the user. When an alert box pops up, the user will have to click "OK" to proceed.
syntex: window.alert("sometext");

Confirm Box

A confirm box is often used if you want the user to verify or accept something. When a confirm box pops up, the user will have to click either "OK" or "Cancel" to proceed. If the user clicks "OK", the box returns true. If the user clicks "Cancel", the box returns false.
syntex: window.confirm("sometext");

Prompt Box

A prompt box allows users to input a value before proceeding. It displays a text field with "OK" and "Cancel" buttons.
syntex: window.prompt("sometext","defaultText");

JavaScript Events

JavaScript Events are actions or occurrences that happen in the browser. They can be triggered by various user interactions or by the browser itself.

Event Types:
  • Mouse Events: click, dblclick, mousemove, mouseover, mouseout
  • Keyboard Events: keydown, keypress, keyup
  • Form Events: submit, change, focus, blur
  • Window Events: load, resize, scroll

JavaScript Event Attributes
Event Attribute Description
onclick Triggered when an element is clicked.
onmouseover Fired when the mouse pointer moves over an element.
onmouseout Occurs when the mouse pointer leaves an element.
onkeydown Fired when a key is pressed down.
onkeyup Fired when a key is released.
onchange Triggered when the value of an input element changes.
onload Occurs when a page has finished loading.
onsubmit Fired when a form is submitted.
onfocus Occurs when an element gets focus.
onblur Fired when an element loses focus.
